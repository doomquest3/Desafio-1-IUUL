CSS, denominado Cascading Style Sheets ou Folha de Estilo em Cascatas, é um mecanismo para adicionar estilo a um documento web, HTML. O código CSS pode ser aplicado diretamente nas tags ficando contido dentro das tags "style". É possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos.
